自我介绍

我个人是被前端各种炫酷的效果吸引过来的，从此爱上前端，惊喜于每一种 CSS 样式所带来的不同效果；学习的初期会看各种效果，思考是怎么样实现的，并去模仿；先后接触 jquery、zepto、bootstrap 等前端库，慢慢也接触了解 less 这样的框架，babel、grunt等工具。因自学过OC，期待前端也能够把 view、controller、model 分开处理，不至于一张页面太过臃肿，此后便接触到 Angular、React、Backbone 之类的前端框架，Backbone用的比较少，某些场景会选择使用，现在的项目会选择使用 React ＋ Flux ＋ webpack 等技术构建。此外，在项目进行中的沟通也算是擅长吧，对于项目原型的逻辑、UI 的设计、后台的接口都能发现问题并从个人角度提出建议。个人也正致力于不让自己仅仅开发前端页面，更深层次了解前端的各方面，正如初心这一理念。

了解一门后端语言 java，node 简单看过文档，没有在实际项目中使用过。不晓得这个自我介绍是不是太长了，原谅我是个女生，嘿嘿～

项目经验

从大三暑假开始出来实习，到现在一年半的时间，真正开始从事前端工作是2015年11月份。现在感觉做的项目都是自己孩子一样的感觉。

温州市人民医院、温州市第七人民医院支付宝服务窗

项目起止时间：2015-11~2016-03
职责：最初从事前端，辅助开发（全部页面的开发以及部分逻辑），给用户提供在线预约挂号的功能，也提供了智能导诊以及一系列业务功能。

眼视光试镜

项目起止时间：2015-03~2016-07
职责：负责这个项目的整个前端开发工作，主要功能是使用 Face++ 的接口数据，自动给相片的人物带上眼镜，用户也可以缩放和拖动调整眼镜的位置，并将数据上传，以及一些辅助功能。

智能家居 APP 页面

项目起止时间：2016-08~2016-09
职责：负责部分项目的开发，在此过程中稍稍了解硬件模块的数据传输，比如主控、透传之类的，嘿嘿～由于不同页面使用的组件都是差不多的，熟悉业务便考虑将按钮之类的组件用 backbone 封装，提升开发效率。

运维管理系统

项目起止时间：2016-09~2016-11
职责：项目前端负责人，开始考虑用 webpack ＋ React ＋ Flux 等开发整个项目。这个项目的最大的收获便是学会思考。
